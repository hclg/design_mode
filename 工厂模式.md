### 场景

1. 同常需要利用子类去创建实例化对象
2. 定义一个接口，由子类去实例化
   1. 子类实现的实例化方法就是工厂方法

> 代码实现：
>
> java 版
>
> ```java
> 
> class Ahh implements A {
>     @Override
>     public String toString() {
>         return "AHH";
>     }
> }
> 
> public interface Factory {
>     public A createA();
> }
> 
> class HCLGFactory implements Factory{
>     public A createA() { // 工厂方法
>         return new Ahh();
>     }
> }
> ```
